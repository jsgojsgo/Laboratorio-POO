# README

**Explicación de la actividad:**
La actividad se enfoca en un programa que permita obtener la salida:
```
Output: 
Televisión: MARCA, Modelo: ¿??????, Color: ¿??? 
Dispositivo Enciende 
Cambia de canal  
Configuración de Menú 
Dispositivo Apagado 
```
Mediante encapsulamiento y creación de distintas clases.

**Requisitos**
- Java 8 o superior
- Compilador de Java (`javac`) y terminal para ejecutar

**Guía para ejecutar**
1. Abrir cualquier archivo `.java`
2. Compilarlo y ejecutarlo
