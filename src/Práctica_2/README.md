# README


**Explicación de la actividad:**
La actividad se enfoca en la familiarización del entorno java y entrada y salida de datos
- Actividad_1.java : Primer 'Hola Mundo'
- Actividad_1_1.java : Programa que lee e imprime cada uno de los tipos de datos primitivos
- Actividad_1_2.java : Programa que pregunta al usuario nombre, edad y número de teléfono, para imprimirlo después


**Requisitos**
- Java 8 o superior
- Compilador de Java (`javac`) y terminal para ejecutar

**Guía para ejecutar**
1. Abrir cualquier archivo `.java`
2. Compilarlo y ejecutarlo
