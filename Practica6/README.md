# README

**Explicación de la actividad:**
La actividad se enfoca en un programa que permite mostrar el siguiente output:
```
Telefono Marca - modelo 
Encender 
Conexcion WIFI  
Iniciar Llamada + Numero 
Iniciar Llamada + Nombre 
Finalizar Llamada 
```
Mediante distintas clases, métodos y encapsulamiento.

**Requisitos**
- Java 8 o superior
- Compilador de Java (`javac`) y terminal para ejecutar

**Guía para ejecutar**
1. Abrir cualquier archivo `.java`
2. Compilarlo y ejecutarlo
