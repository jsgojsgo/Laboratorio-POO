# README

**Explicación de la actividad:**
La actividad se enfoca en un programa que permite:
- Retornar longitud de un `String`
- Determinar si dos cadenas son iguales
- Ordenar un arreglo
- Determinar el mayor de 2 y 3 números
- Retornar el IVA dados la cantidad de producto y precio
- Dividir una cadena caracter a caracter

**Requisitos**
- Java 8 o superior
- Compilador de Java (`javac`) y terminal para ejecutar

**Guía para ejecutar**
1. Abrir cualquier archivo `.java`
2. Compilarlo y ejecutarlo
