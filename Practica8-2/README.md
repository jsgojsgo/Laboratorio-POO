# README

**Explicación de la actividad:**
La actividad se enfoca en un programa que permite hacer uso de:
```
@Override
@Deprecated
@Documented
@CustomAnnotation
```
Para obtener propiedades y métodos de un automóvil

**Requisitos**
- Java 8 o superior
- Compilador de Java (`javac`) y terminal para ejecutar

**Guía para ejecutar**
1. Abrir cualquier archivo `.java`
2. Compilarlo y ejecutarlo
