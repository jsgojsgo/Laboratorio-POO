# README

**Explicación de la actividad:**
La actividad se enfoca en la resolución de problemas con operaciones básicas. Dados dos números, se realiza las operaciones básicas de suma, resta, multiplicación y división

**Requisitos**
- Java 8 o superior
- Compilador de Java (`javac`) y terminal para ejecutar

**Guía para ejecutar**
1. Abrir cualquier archivo `.java`
2. Compilarlo y ejecutarlo
