# README

**Explicación de la actividad:**
La actividad se enfoca en un programa que permite definir un `ArrayList` de 10 automóviles, que contengan sus respectivas propiedades, además de mostrar el resultado

**Requisitos**
- Java 8 o superior
- Compilador de Java (`javac`) y terminal para ejecutar

**Guía para ejecutar**
1. Abrir cualquier archivo `.java`
2. Compilarlo y ejecutarlo
