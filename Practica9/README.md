# README

**Explicación del proyecto:**
El proyecto consiste en un sistema de manejo de archivos en Java, que permite:

- Crear, leer, modificar y eliminar archivos de texto
- Crear y leer archivos binarios
- Crear y eliminar directorios
- Leer datos desde un archivo CSV y exportarlos a un archivo TXT (formato alternativo)
- Guardar backups automáticos de archivos modificados con timestamps
- Registrar un historial de acciones realizadas

Se eligió exportar a TXT por la facilidad de lectura y compatibilidad con cualquier editor de texto

**Requisitos**
- Java 8 o superior
- Compilador de Java (`javac`) y terminal para ejecutar

**Guía para ejecutar**
1. Abrir el archivo `Practica9.java` (contiene el `main`)
2. Compilarlo y ejecutarlo (a partir de aquí, el código realiza todos los procesos por su cuenta)
